# Medical Insurance Cost Prediction

This project is a Streamlit web app that analyzes a medical insurance dataset and predicts insurance charges using machine learning models.

## Project Files

- `medical/Medical.py`: Streamlit application and machine learning model code.
- `insurance.csv`: Dataset used for training and analysis.
- `requirements.txt`: Required Python libraries.

## Dataset Columns

The dataset contains the following columns:

- age
- sex
- bmi
- children
- smoker
- region
- charges

## Models Used

- Linear Regression
- Lasso Regression
- Ridge Regression
